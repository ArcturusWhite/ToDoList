# TodoList

TodoList is a mobile-first PWA task manager built with Next.js 15.3.6 and Tailwind CSS. It is designed for professional work tracking with a clean interface inspired by Todoist and Notion.

## Features

- Pending and In Progress task tabs
- Create, edit, delete, complete, expand, and collapse tasks
- Critical, Intermediate, and Low priority levels
- Automatic sorting by completion status and priority
- Completed task styling with gray, strikethrough, and reduced opacity
- LocalStorage persistence after refresh
- Statistics page with totals and priority breakdown
- Mobile-first responsive layout with bottom navigation
- PWA manifest, service worker, and app icons

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
